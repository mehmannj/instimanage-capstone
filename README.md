InstiManage is a full-stack smart campus management platform developed as a capstone project.  
The platform enables educational institutions to efficiently manage campus resources such as lockers and rooms through a centralized digital system.

---

## Project Overview

Many institutions rely on manual or fragmented processes to manage campus resources. InstiManage solves this problem by providing a unified platform where users can book resources while administrators manage availability, usage, and reporting through a secure system.

---

## Key Features

- Locker and room booking system  
- Secure authentication and role-based access  
- Administrative dashboards and controls  
- Reporting and monitoring features  
- REST-based backend architecture  
- Modular and scalable design
